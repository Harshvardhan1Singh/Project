# Keeper
An Online note taking app<br/>

For running in your system:</br>

Create a .env file in root directory</br>

Include:</br>

NODE_ENV=</br>
MONGO_URI=</br>
TOKEN_SECRET=<br/></br>

Then </br>

cd Keeper</br>
npm install</br>
npm start</br>
</br>

For client

cd Keeper/client</br>
npm install</br>
npm start</br>
